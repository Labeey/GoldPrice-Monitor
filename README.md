# GoldPrice-Monitor
 Displays gold price in realtime using ESP32 and Oled display 
 
- Connects to Wifi
- Fetches Gold price using JSON from gold-api.com (no need for API KEY)
- Displays gold price on OLED.

 **Features**
 
 - Creates AP for easy SSID and password setup
 - Fetches GOLD, Silve, Platinum, Palladium price.
 - Multiple currencies: USD, Euro, GBP, etc.
 - Only $5 in parts.
 - 5 minutes to setup
 
**Order parts from Amazon:**
   
   OLED SSD1306 $9/3pcs https://amzn.to/3MppA2w
   Esp32 $11.99/3pcs https://amzn.to/3MLz08m

**Programming:**

   - Using Arduino IDE select "ESP32C3 Dev Module"
   - Press and HOLD "BOOT" button and tap "RST" button once 
   - Copy and paste the code
   - Compile and Upload
   
**Libraries required**

   - Adafruit SSD1306
   - Adafruit GFX
   - WiFiManager
